# Plan 7.1: Setup Supabase Database Schema and Data Synchronization logic

## Objective
Enable seamless database persistence using Supabase as the primary cloud database, replacing pure local storage. Ensure a local-first approach where data is cached in localStorage, synchronizes in the background when online, and gracefully merges existing local data on sign-in so no user data is lost.
