# Plan 6.3: Dream Board (Quadro dos Sonhos)

## Objective
Implement a Dream Board (Quadro dos Sonhos) tab allowing users to organize their goals/dreams into categories (e.g., Carreira, Viagem, Saúde, Bens, Outros), select custom visual emoji icons, track status ("Em Progresso" or "Conquistado 🏆"), and trigger a celebratory reward animation (gamification) when a dream is conquered.

## Context
- src/types/index.ts
- src/hooks/useBujoItems.ts
- src/context/BujoContext.tsx
- src/features/planner/components/Sidebar.tsx
- src/App.tsx
