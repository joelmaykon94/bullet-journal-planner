# Plan 9.1: Daily Log Features & UI Tweaks

## Objective
Implement crucial Daily Log features and bug fixes:
1. Always reset form inputs (including resetting date to today and clearing time) after task creation.
2. Allow updating task date and time inside the edit form of the task card.
3. Add a filter to list tasks that are pending ("a realizar") vs completed ("realizadas").

## Context
- src/features/planner/components/DailyLogTab.tsx
- src/features/planner/components/BulletItem.tsx
- src/hooks/useBujoItems.ts
- src/context/BujoContext.tsx
