# Plan 9.6: PWA Install & Offline Awareness

## Objective
Provide a native installation experience by leveraging the PWA `beforeinstallprompt` event and improve user awareness of their network status.

## Context
- `src/context/BujoContext.tsx`: Store the deferred prompt and network status.
- `src/App.tsx`: Listen for the window events.
- `src/features/planner/components/Header.tsx`: Trigger the actual install prompt.
