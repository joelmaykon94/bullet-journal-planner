# Plan 9.4: Global Data Management & Technical Debt Removal

## Objective
Enhance the application's data robustness and clean up redundant code by:
1. Consolidating redundant task saving logic between `BujoContext.tsx` and `useBujoItems.ts`.
2. Implementing Global Data Export/Import (JSON) for backups.
3. Implementing Task Export to CSV for external use.
4. Adding a "Clear All Data" action to the settings.
5. Updating the Settings UI to expose these new data management features.

## Context
- `src/context/BujoContext.tsx`: Main context where state and some handlers reside.
- `src/hooks/useBujoItems.ts`: Hook where the primary `handleSaveStandardInput` logic resides.
- `src/features/settings/components/SettingsTab.tsx`: UI for data management actions.
