# Plan 9.5: Global Search & Omnibox

## Objective
Implement a global search feature that allows users to find items (tasks, events, notes) across the entire application history, addressing the limitation of the current date-scoped search.

## Context
- `src/context/BujoContext.tsx`: Need state for the search modal and global items.
- `src/features/planner/components/Header.tsx`: Location for the search trigger button.
- `src/features/planner/components/GlobalSearchModal.tsx`: New component for the search interface.
