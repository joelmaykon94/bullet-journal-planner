# Plan 13.1: Contextual Feature Help & Floating Button

## Objective
Implement a contextual help system that shows specific usage tutorials the first time a user visits a feature tab, and allows on-demand access via a floating "Help" button.

## Context
The user requested that each feature (Daily Log, Weekly Log, Future Log, Brain Dump, etc.) have its own specific "How to use" tutorial. This should trigger on the first visit to that feature, similar to the general tour, but focused. It should also be manually triggerable via a floating help icon.

## Tasks

- [ ] **Task 1: Add Help Content Dictionary & State**
  - Type: auto
  - Description: Create a FeatureHelpConfig dictionary mapping each activeTab to its help content (Title, Description, How to use, Shortcuts). Add state viewedFeatureHelp: Record<string, boolean> to BujoContext, synced to localStorage. Add showFeatureHelpModal state.
  - Files: src/context/BujoContext.tsx

- [ ] **Task 2: Create FeatureHelpModal.tsx Component**
  - Type: auto
  - Description: Create a new modal component that reads the current activeTab, pulls the relevant help content, and displays it. It should have a Got it button that closes it.
  - Files: src/components/common/FeatureHelpModal.tsx

- [ ] **Task 3: Integrate Floating Button & Auto-trigger**
  - Type: auto
  - Description: Inside AppContent render the FeatureHelpModal. Add a floating ? Help icon fixed to the bottom-left (bottom-6 left-6) to avoid clashing with the FAB (bottom-right) and Mascot (bottom-right). Add a useEffect inside AppContent that watches activeTab. If the current tab hasnt been viewed according to viewedFeatureHelp, set it to true (save to localStorage) and open the FeatureHelpModal.
  - Files: src/App.tsx

## Verification Criteria
1. Visiting a tab like brain_dump for the first time opens the specific Help Modal for it.
2. Clicking Got it closes the modal.
3. Refreshing and visiting the same tab again does NOT auto-open the modal (persisted state).
4. Clicking the floating ? button manually opens the Help Modal for the current feature context.
