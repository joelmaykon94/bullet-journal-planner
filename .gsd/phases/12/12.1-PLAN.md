# Plan 12.1: Todoist-style Smart Add NLP Parser

## Objective
Implement an advanced Natural Language Processing (NLP) parser for task inputs, inspired by Todoist, to automatically extract dates, times, and priorities, while keeping the task description clean.

## Context
The user wants a faster way to input tasks by typing natural sentences like "Reunião amanhã às 15h p1". The system should intelligently parse this, set the internal task metadata (date, time, priority, energy, complexity), and strip the triggering keywords from the final text. 

## Tasks

- [x] **Task 1: Create `smartParser.ts` utility**
  - Type: auto
  - Description: Implement the `parseSmartTask` function in `src/utils/smartParser.ts`. It must use regex to identify and remove dates (hoje, amanhã, dias da semana, "daqui a X dias"), times (às 15:00, 15h), and priorities (p1, p2, p3, p4).
  - Logic rules:
    - `p1`: priority=true, energy=5, complexity=5
    - `p2`: priority=true, energy=4, complexity=4
    - `p3`: priority=false, energy=3, complexity=3
    - `p4`: priority=false, energy=2, complexity=2
  - Files: `src/utils/smartParser.ts`

- [x] **Task 2: Integrate Smart Add into `useBujoItems.ts`**
  - Type: auto
  - Description: Update `handleSaveStandardInput` to pass the `standardInput` through `parseSmartTask` before creating the task. Override the manual states (date, time, energy, complexity) if the parser detects them.
  - Files: `src/hooks/useBujoItems.ts`

- [x] **Task 3: Add visual hint to input field (DayTasksModal & DailyLogTab & SomedayMaybeTab)**
  - Type: auto
  - Description: Update the input placeholders in the app to hint at the new capability. E.g., `Adicionar tarefa (ex: "Revisar docs amanhã às 14h p1")`.
  - Files: `src/features/planner/components/DayTasksModal.tsx`, `src/features/planner/components/DailyLogTab.tsx`, `src/features/planner/components/SomedayMaybeTab.tsx`

## Verification Criteria
1. Typing "Tarefa p1" results in a task named "Tarefa", priority=true, energy/complexity=5.
2. Typing "Reunião amanhã às 10h" sets the task date to tomorrow, time to 10:00, and leaves text as "Reunião".
3. Placeholders in input fields reflect the new capability.
