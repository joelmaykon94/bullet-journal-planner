# Plan 14.2: Task Links Input and Clean Domain Rendering

## Objective
Add a collapsible text input field in all task creation forms to capture a link (URL) with the task, and render the link as a small, clean domain hostname badge inline instead of displaying the raw URL.

## Tasks
- [x] **Task 1: Add link property to BujoItem interface**
  - Files: `src/types/index.ts`
- [x] **Task 2: Support link parameter in context and hooks**
  - Files: `src/context/BujoContext.tsx`, `src/hooks/useBujoItems.ts`
- [x] **Task 3: Add collapsible link input to task creation forms**
  - Files: `src/features/planner/components/DailyLogTab.tsx`, `src/features/planner/components/DayTasksModal.tsx`, `src/features/planner/components/FutureLogTab.tsx`, `src/features/planner/components/MonthlyLogTab.tsx`, `src/features/planner/components/SomedayMaybeTab.tsx`
- [x] **Task 4: Render clean domain links in task lists**
  - Files: `src/features/planner/components/BulletItem.tsx`, `src/features/planner/components/SomedayMaybeTab.tsx`

## Verification Criteria
1. Project compiles without TypeScript errors.
2. Link inputs are collapsible (open by default in Daily Log, closed in others).
3. Links render as short, clickable domain hostnames.
